# Escucha tu Historia — Mi parte

## Archivos desarrollados por mí

### Home
- `home_screen.dart` — Pantalla principal con hero, menú de navegación y monumentos recomendados

### Settings
- `settings_screen.dart` — Pantalla de ajustes (modo oscuro, idioma, notificaciones, contenido offline)
- `licencias_screen.dart` — Pantalla de licencias de código abierto
- `privacidad_screen.dart` — Pantalla de política de privacidad
- `config_provider.dart` — Provider que gestiona y persiste los ajustes del usuario

### Favoritos
- `favorites_service.dart` — Servicio que guarda y lee favoritos en disco con SharedPreferences
- `favoritos_provider.dart` — Provider que expone el estado de favoritos a la UI
- `favoritos_screen.dart` — Pantalla de favoritos con búsqueda local
